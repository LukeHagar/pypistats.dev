About PyPI Stats

Goal

PyPI Stats aims to provide aggregate download information on python packages available from the Python Package Index.

Data

Download stats are sourced from the Python Software Foundation's publicly available download stats on Google BigQuery. All aggregate download stats ignore known PyPI mirrors (such as bandersnatch) unless noted otherwise.

PyPI Stats attempts to operate within the free tier in AWS. For this reason, aggregate data is only retained for 30 days.

Tech

PyPI Stats is a project developed using Python 3.6.

  • Framework: Flask
  • Host: AWS
  • Deployment: Zappa
  • Authentication: GitHub OAuth
  • ORM: SQLAlchemy
  • DBAPI: psycopg2
  • RDBMS: alembic
  • Templating: jinja2
  • Charts: plotly.js
  • Data: Google Cloud's BigQuery
  • And many more open source software packages

Who

PyPI Stats was created by Christopher Flynn.
